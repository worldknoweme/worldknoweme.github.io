第七节JavaScript基础（一）

发表于 2018-09-14 | 分类于 web基础
字数统计: 2,208 | 阅读时长 ≈ 10

说明

平时我们说的js就是JavaScript.
为啥叫js呢?因为对于的JavaScript文件都是js结尾的.
js主要由语言基础 bom和dom组成.
首先,通过一个例子感受一下js的作用.

有人可能会说,以上仅仅是显示一段文字,我完全可以使用html标签来实现啊.
但是html标签是死的,就是说我只能存入一个常量,写死.而js呢,可以将变量进行输出,使得页面变成活的,可变的.
所以说呢,html又叫做静态网页.
js呢,则是客户和程序交互的一种脚本语言,因为js的存在,使得静态网页动了起来.

dom

dom是啥呢?简单来说,就是html元素的一种层级关系,通过dom呢,可以方便的找到我们关注的元素.

总体结构

script标签

使用script标签来包围js代码,在该标签内的代码,html才会认为这是一段js代码,按照js的逻辑去执行.
script标签是html的一个标签
看一下刚开始的第一个程序,通过js向网页上输出一段字符串.

js顺序执行

需要注意js的代码是顺序执行的,和java类型,一行一行的执行.

使用外部js

同css类型,我们可以将js代码放入一个指定的js文件里面,之后将文件引入到我们的html网页里.
网页里面去执行js代码.
<html> <script src="/hello.js"></script> </html>

js注释

主要分为两种,单行注释和多行注释.
和java注释相同,单行使用//多行使用/*

变量

使用var来声明一个变量.不需要指定变量类型.

js的写法比较天马星空,这主要得益于浏览器的强大.比如,不是有var去声明一个变量,浏览器也可以进行解析.

调试

  1. 使用chrome进行代码调试
  2. 使用alert进行弹框调试
  3. 使用console.log()进行控制台信息输出调试

类型转换

使用toString()方法进行转字符串操作.

数字转字符串,主要分为默认模式和基模式转换.默认转换为十进制,基模式可以转为其他进制.

转数字,使用parseInt()和parseFloat()

转波尔,使用Boolean()即可.

Number()和parseInt()的区别

Number的话,只要转换的数据存在非数字的元素,则会返回NAN.
parseInt的话,数字开头的会返回开头数字,非数字开头才会返回NAN.

String()和toString()的区别

String()和toString()一样都会返回字符串，区别在于对null的处理
String()会返回字符串”null”
toString() 就会报错，无法执行

函数

function关键字用于定义一个函数
print即函数的名称
()表示参数列表，像这样就指没有参数
{ 表示函数开始
} 表示函数结束

==

其实，==作用于基础数据类型，比如int、long、float等。

说到对象，在进行比较的时候如果使用==运算符，那么比较的就不仅仅是对象的值了，还需要比较对象的内存地址。因为上述例子中的两个string均是new出来的一个对象，也就是说对于的内存地址是不一致的，因此得到的结果是false。
